# QA Automation Framework

A Java Selenium automation framework built using TestNG, Maven, and the Page Object Model (POM) design pattern.

## Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Log4j2 Logging
- Extent Reports
- Jenkins CI

## Features

- Page Object Model architecture
- XML test execution with TestNG
- Logging with Log4j2
- Extent HTML reporting
- External test data support (Excel)
- Maven project structure

## Project Structure

```
src/
 ├── main/
 │    ├── base/
 │    ├── pages/
 │    ├── utils/
 │    └── resources/
 └── test/
      └── tests/
```

## Run the Tests

```bash
mvn test
```
